## MIRA-DB EXPRESS DATABASE (INSTALL)
---
### 1) INSTALL MIRA
``` 
npm install mira-db 
```

### 2) INSTALL EXPRESS
```	
npm install express 
```

### 3) INSTALL BODY PARSER
```	
npm install body-parser 
```

---

### SET YOUR STORAGE FOLDER ## 
```js
 "mira_express_database.js" Line:8

 var DB_DIR = "./your_storage_folder"; 
```

### QUERY (GET,POST)
``` 
http://localhost:8123/query 
```
### PUBLIC HTTP
``` 
http://localhost:8123
```
