# MIRA-DB EXPRESS DATABASE (HTTP API)
	
### INSTALL MIRA
``` npm install mira-db ```

### INSTALL EXPRESS
```	npm install express ```

### INSTALL BODY PARSER
```	npm install body-parser ```
		
### GET,POST URL (TYPE)
Content-type: application/x-www-form-urlencoded

```http://localhost:8123/query```
