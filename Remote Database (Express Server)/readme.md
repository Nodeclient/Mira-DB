# MIRA-DB EXPRESS DATABASE (HTTP BASED)
	
### INSTALL MIRA
``` npm install mira-db ```

### INSTALL EXPRESS
```	npm install express ```

### INSTALL BODY PARSER
```	npm install body-parser ```
		
### GET,POST URL
``` http://localhost:8123/query ```
