### Sample DB - NPM package Installation : @1.0.40

    run this command in your terminal:
### Remote-DB Install
    $ 'cd ./mira-remote'
    $ 'npm install'
    $ 'npm run mira_local_database.js'

### Local-DB Install
    $ 'cd ./mira-local'
    $ 'npm install'    
    $ 'npm run mira_db_express.js'
